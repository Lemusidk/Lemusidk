## 🚀 Proyectos Destacados

### 🧪 Detección de Anomalías en Consumo Automotriz
Modelo de **Machine Learning no supervisado (Isolation Forest)** para identificar vehículos ineficientes.

**Resultados clave:**
- Identificación del 5% de la flota como anómala
- Análisis de correlación entre potencia (HP) y emisiones de CO₂
- Enfoque en detección temprana de ineficiencias operativas
